Uses your site logo (defined through Jetpack) or your site icon as the logo on the login form.

== Description ==
WordPress allows you to define a site logo (through Jetpack) and a site icon, why would you need to upload **another** image to use in your login page?

This plugin allows you to forget the login page. Just activate it and all the magic is done for you. No extra steps required.

The plugins uses the site logo if it exists, otherwise it will use the site icon. If none of them exist, the plugin will not touch your login page.

== Installation ==
1. Install the plugin either via the WordPress.org plugin directory, or by uploading the files to your server.
2. Activate it.
3. No third step. It's ready!

== Frequently Asked Questions ==
= Do I need Jetpack to use this plugin? =

No. If you don't have Jetpack, the plugin will use the site icon.
